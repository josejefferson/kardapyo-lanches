## Funcionalidades

- CRUD de produtos
- Pesquisa de produtos
- Autenticação
- Proteção CSRF
- Sessões
- HTML Templates
- Mensagens flash
- Migrações de banco de dados
- Validação

## Instalação

**Requisitos:** Node.js e Yarn (`npm install -g yarn`)

```sh
yarn install
```

## Executar

**Desenvolvimento:** `yarn dev`

**Produção:** `yarn start`
